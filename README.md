## Eagle Owl

### Demo link: "https://hi-malay.github.io/eagleowl/"


## Steps to run front-end:
1. git clone link 
2. cd template
3. npm install
4. npm start
5. go to: http://localhost:3000/main/login on browser

eagelowl app lets you:
1. Select Table rows
2. Search in table
3. Sort in table
4. Paginate through various feild
5. Show the best data among table
